This repository contains exercises and experiments for the Linux Bash Scripting Lab. The purpose is to learn Linux command-line basics, shell scripting, file and storage management, process management, and other essential Linux administrative tasks.

The repository is organized so that each experiment has its own folder containing:
  
  1) README.md – Experiment report.
  2) .pdf – Experiment pdf files.
  3) screenshots/ – Screenshots of outputs and results.
  4) config.txt – Commands used in the experiment.

Repository Structure:

Linux-Bash-Scripting-Lab/
│
├── Experiment-1/
│   ├── README.md
│   ├── Experiment 1.pdf
│   ├── config.txt
│   └── screenshots/
│
├── Experiment-2/

│
└── Experiment-n/
    ├── README.md
    ├── Experiment n.pdf        # n is a number of experiment
    ├── config.txt
    └── screenshots/

  How to Use:

  1) Clone the repository:
    . https://github.com/Aman12-security/Linux-Bash-Scripting-Lab-Aman-Singhal.git
    . cd Linux-Bash-Scripting-Lab

  2) Open the folder for the experiment you want.
  3) Run commands/scripts or follow the steps in the README.md.
  4) Check screenshots for verification of results.

Learning Outcomes:

  1) Execute Linux commands and Bash scripts.
  2) Manage files, processes, and storage devices in Linux.
  3) Understand mounting/unmounting and directory structures.
  4) Document experiments professionally using GitHub.
